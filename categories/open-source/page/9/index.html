Thu, Sep 20, 2007 This release of MySQL Toolkit adds a new tool, fixes some minor bugs, and adds new functionality to several of the tools. New tool: MySQL Heartbeat This tool was contributed by Proven Scaling’s Jeremy Cole and Six Apart. It measures replication delay on a replica, which can be daisy-chained to any depth. It does not rely on SHOW SLAVE STATUS, and in fact it doesn’t even need the replica processes to be running.
