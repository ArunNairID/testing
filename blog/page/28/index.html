Tue, Apr 28, 2009 Peter wrote about this recently, but I don’t know if it was really clear what was going on. Point One: Sphinx can be contacted by the MySQL protocol. Not “as a MySQL storage engine.” Not “from MySQL.” It understands the MySQL protocol itself. So from the protocol point of view, the Sphinx search daemon can look just like a MySQL server.
