How to write SQL JOIN clauses more compactly

Tue, Nov 28, 2006 Here’s a way to write join clauses so they are more compact, more readable, and less confusing. Standard SQL:2003 defines a USING clause that can serve the same function as the ON clause in the familiar JOIN syntax. For example, the following join select a.col1, b.col2 from a inner join b on a.col3 = b.col3 May be written as follows: select a.col1, b.col2 from a inner join b using(col3) That may not look like much of an improvement, but it is a big help in larger joins where many tables have columns with the same names.
